# Dynamic Profile Cards App

A modern React + Next.js profile card application built using Props and State concepts.

## Features

- Dynamic Profile Cards
- Follow / Unfollow Button
- Dark Mode Toggle
- Search User Functionality
- Card Flip Animation
- Responsive Design
- Component-Based Architecture
- React Hooks (useState, useEffect)

## Technologies Used

- React.js
- Next.js
- TypeScript
- CSS3
